# Pizza App

A modern pizza ordering web app with dark mode support built with React and Vite.

## Tech Stack
- React, Vite, Tailwind CSS

## Features
- Pizza menu browsing
- Dark mode toggle
- Responsive design

## Getting Started
```bash
npm install
npm run dev
```
